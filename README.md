## Installation

Use the `cargo add` command to automatically add it to your `Cargo.toml`
```bash
cargo add mcp_core
```
Or add `mcp_core` to your `Cargo.toml` dependencies directly
```toml
[dependencies]
mcp_core = "0.0.9"
```
